# Helmet Detection using YOLOv11

## Project Overview

This project develops a helmet detection model using YOLOv11 for construction and industrial safety applications.

## Dataset

* Total Images: 614
* Classes: 1 (Helmet)
* Train Images: 374
* Validation Images: 49
* Test Images: 191

## Model

* YOLOv11 Small
* Image Size: 640 × 640
* Epochs: 50
* Framework: Ultralytics YOLO

## Results

| Metric    | Value |
| --------- | ----- |
| Precision | 99.9% |
| Recall    | 100%  |
| mAP@50    | 99.5% |
| F1 Score  | 99.5% |
